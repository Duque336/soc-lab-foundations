# Lab 01 — Network Observation

## Objective
Build first-principles understanding of network telemetry by:
1) capturing baseline traffic,
2) generating controlled recon traffic (Nmap),
3) comparing how "normal" differs from scanning patterns.

## Success Criteria
- Explain what a TCP SYN scan looks like at packet level.
- Describe the difference between baseline background traffic and recon traffic.
- Produce artifacts (pcap and/or screenshots) and written findings.
